# See Weather

You needs those packages to start:

-   [NodeJs](https://nodejs.org/en)
-   [Angular Cli](https://angular.io/cli)
-   [NestJs](https://nestjs.com/)

To start install all dependencies in every folder when you see package.json
by typing

```
    npm install
```

and after that type

```
    npm run dev
```

in main folder
